# AI-Powered Transport Incident Management using ServiceNow

## Project Overview

This project demonstrates an AI-assisted Incident Management solution built on the ServiceNow ITSM platform for a transport and logistics organization.

The solution focuses on improving operational efficiency by helping the service desk automatically classify transport-related incidents, recommend priorities, suggest assignment groups, and provide knowledge-based resolution recommendations.

Instead of traditional IT incidents such as password resets or Outlook issues, this project is based on real transport operations like vehicle breakdowns, cargo delays, route deviations, driver safety alerts, and fleet management.

---

## Problem Statement

Transport companies handle numerous operational incidents every day. Manually identifying the correct category, priority, and support team increases response time and affects business operations.

This project demonstrates how AI-assisted incident management can support faster incident triage and resolution within ServiceNow.

---

## Objectives

- Improve incident classification
- Reduce incident response time
- Recommend appropriate assignment groups
- Suggest relevant knowledge articles
- Identify recurring incidents
- Improve operational visibility through dashboards
